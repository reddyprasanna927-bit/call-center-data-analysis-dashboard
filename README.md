# Call-center-data-analysis-dashboard


# project overview

This project is an Excel-based interactive dashboard built to analyze call center customer sentiment data. It helps visualize customer feedback trends, agent performance, and overall service quality using clean and interactive Excel charts.


# Dataset
<a href="https://github.com/reddyprasanna927-bit/call-center-data-analysis-dashboard/blob/main/Call-Center-Sentiment-Sample-Data.xlsx">data</a>


# Questions(kpi)
What percentage of calls are Positive, Negative, and Neutral
Which sentiment is most common among customers
How many total calls were received in a given period
What is the average number of calls per day/week
Which agent handled the most calls
Which agent has the highest resolution rate
What is the average call duration
What is the overall call resolution rate (%)
What are the most common customer issues
