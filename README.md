# Call-center-data-analysis-dashboard


# project overview

This project is an Excel-based interactive dashboard built to analyze call center customer sentiment data. It helps visualize customer feedback trends, agent performance, and overall service quality using clean and interactive Excel charts.


# Dataset
<a href="https://github.com/reddyprasanna927-bit/call-center-data-analysis-dashboard/blob/main/Call-Center-Sentiment-Sample-Data.xlsx">data</a>


# Questions(kpi)
-What percentage of calls are Positive, Negative, and Neutral
-Which sentiment is most common among customers
-How many total calls were received in a given period
-What is the average number of calls per day/week
-Which agent handled the most calls
-Which agent has the highest resolution rate
-What is the average call duration
-What is the overall call resolution rate (%)
-What are the most common customer issues

# Dashboard intraction

# Process
-Total calls and customer sentiment distribution are analyzed.
-Agent performance is measured based on resolution rate and call handling.
-Issue types are identified to understand common customer problems.
-Overall dashboard helps track call center efficiency and satisfaction.

# Project Insights
-The analysis shows that customer sentiment is mostly positive and neutral, indicating overall satisfactory service levels.
-Certain agents consistently perform better in terms of resolution rate and customer feedback.
--A few issue types contribute significantly to negative sentiment and require process improvement.
-Call volume trends highlight peak activity periods that can help optimize workforce planning.
-Resolution rate and call duration analysis help identify efficiency gaps in handling customer queries.
# Final Conclusion

The Call Center Sentiment Dashboard provides meaningful insights into customer behavior, agent performance, and operational efficiency. It helps identify strengths such as high-performing agents and areas that need improvement like recurring issue types and negative feedback trends. By using these insights, organizations can improve customer satisfaction, reduce unresolved calls, and optimize overall call center performance. This dashboard transforms raw data into clear, actionable business decisions.
